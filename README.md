# Lean Six Sigma Process Business Improvement Analysis

## Project Status

Completed Academic OJT Project

## Project Overview

This project was completed as part of an On-the-Job Training (OJT) program focused on Lean Six Sigma methodology, business process improvement, and data-driven decision making in the financial services sector.

The objective was to analyze customer preferences for personal loan products and identify the most influential factors affecting customer decision-making using survey-based data collection and predictive analytics.

## Business Problem

Financial institutions operate in a highly competitive market where understanding customer preferences is essential for designing attractive loan products.

The challenge was to identify the loan features that customers value the most and recommend optimized product configurations that improve customer satisfaction and business performance.

## Objectives

* Collect customer preference data using structured surveys
* Analyze key factors influencing personal loan selection
* Apply statistical analysis techniques to evaluate customer responses
* Identify high-priority loan attributes
* Generate business recommendations for product optimization
* Support data-driven decision making using predictive analytics

## Methodology

The project was executed using a two-phase survey-based analytical framework.

### Phase 1
- Designed and distributed a structured questionnaire
- Collected responses from approximately 100 participants
- Converted survey responses into binary (0-1) format using Microsoft Excel
- Performed predictive analytics to identify the most influential personal loan selection factors

### Phase 2
- Developed a second questionnaire based on Phase 1 findings
- Collected responses from approximately 200 participants
- Processed and analyzed data using AI Predictive Analytics tools
- Identified optimized personal loan feature combinations preferred by customers

## Key Findings

### Phase 1 Results
- Test Statistic: 54.746
- P-Value: 0.00000
- Assurance Level: 100%

Most influential factors identified:

1. F2 – Disbursal Time
2. F5 – Documentation
3. F3 – Foreclosure Facility
4. F6 – Repayment Tenure
5. F8 – Insurance Benefits
6. F7 – Interest Structure

### Phase 2 Results

The predictive analytics model evaluated 4,096 possible combinations of loan features and identified the optimal customer-preferred product configuration.

Best-performing combination:

A2 B3 C2 D2 E2 F2

## Business Impact

The study provides actionable insights for financial institutions to:

- Design customer-centric loan products
- Improve approval and disbursal processes
- Offer flexible repayment options
- Enhance customer satisfaction
